About

Hello, I'm Kaio Rosa. I'm a web developer who loves teaching and creating. What I love about coding is that the possibilities are endless, you can automate your home, create your own game or even your own business. Currently I've been using a lot of Angular and Node but I'm language agnostic. I see code as a tool to solve problems and I find solving problems very intriguing and that's what made want to be a software developer in the first place.

When I'm not coding I'm recording videos, reading or playing Sudoku on my phone.

Some fun facts about me:

  • I have a twin sister.
  • I learned how to touch type in 2019.
  • I love learning languages. I can speak three languages fluently, English, Spanish and Portuguese, my native language. I'm learning French at the moment.
  • I did 6 rounds of the 100 Days of Code Challenge
